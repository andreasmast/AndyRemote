Andy Remote

Andy Remote is a university project within the lecture Mobile Systems, where students had to build up a remoteproject. The basic idea was that you can control something with your smartphone via a Raspberry Pi. I decide that a want to build a car which i can steer by my smartphone. Therefore we needed a Raspberry Pi on which a web-server runs.

All in all a crazy project like deadpool!
